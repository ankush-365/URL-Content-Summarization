# 🦜 URL Content Summarizer (YouTube + Website)

A Streamlit app that summarizes content from:

- 📺 YouTube videos (via transcript extraction)
- 🌐 Websites (via web scraping)

Built using **LangChain + Groq (LLaMA 3.1) + Streamlit**

---

## 🚀 Features

- Extracts YouTube video transcripts automatically
- Summarizes long videos into structured bullet points
- Scrapes and summarizes website content
- Uses Groq LLaMA 3.1 for ultra-fast inference
- Clean Streamlit UI
- Works locally and on Streamlit Cloud

---

## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- Groq API
- youtube-transcript-api
